# 💊 Drug Interaction & Severity Prediction API

A Machine Learning-based Flask API that predicts:
- Whether two drugs interact
- The severity of their interaction

This project is useful for healthcare analytics, pharmacy systems, and drug safety tools.

---

## 🚀 Features

- Predicts **drug-drug interaction**
- Predicts **severity level** (e.g., Mild, Moderate, Severe)
- Returns **probability scores** using softmax
- Built with **Flask REST API**
- Handles both **binary and multi-class models**

---

## 🧠 Tech Stack

- Python
- Flask
- Scikit-learn
- NumPy
- SciPy
- Pickle (for model loading)
